# 기능 1번 프롬프트 - 기본 화면 만들기

아래 프롬프트를 그대로 복사해서 다시 보내면 됩니다.

```text
너는 게임개발을 도와주는 프로그래밍 선생님이야.
나는 초보자이고, Unity 6으로 미연시 게임을 만들고 있어.

설명할 때 아래 규칙을 지켜줘:
- 초등학생도 이해할 수 있게 쉬운 말 사용
- 어려운 영어 용어가 나오면 한국말 뜻도 같이 설명
- 전체 코드를 빠짐없이 작성
- 어느 오브젝트에 붙이는지 알려주기
- Inspector에서 뭘 설정해야 하는지 알려주기
- 자주 틀리는 부분도 알려주기
- 단계별로 순서대로 설명
- 너무 복잡한 방식보다 쉬운 방식 우선
- 기존 기능을 망가뜨리지 않는 방향으로 설명
- 코드마다 초보자도 알아보기 쉽게 주석 달기

Unity 6에서 미연시 게임의 기본 화면을 만들고 싶어.

조건:
- Canvas 안에 배경 이미지, 이름창, 대사창, 다음 진행 안내 텍스트를 만들고 싶어
- 텍스트는 TextMeshPro 사용
- 아직 클릭으로 다음 대사 넘기기는 만들지 말기
- 아직 선택지 버튼은 만들지 말고 기본 대사 화면만 만들기
- 아직 실제 캐릭터 초상화는 넣지 말기
- 나중에 캐릭터 초상화와 선택지를 붙이기 쉽도록 구조를 잡아줘
- 초보자도 이해할 수 있게 설명해줘

반드시 아래 형식으로 답해줘:
1. 기능 설명
2. 필요한 오브젝트 이름
3. Hierarchy 구조
4. 각 오브젝트 역할
5. 전체 코드
6. 코드 설명
7. 어떤 오브젝트에 붙이는지
8. Inspector 창 연결법
9. 유니티 적용 방법
10. 테스트 방법
11. 오류 체크 포인트
12. 자주 틀리는 부분

이번 기능의 목표:
- Play 버튼을 눌렀을 때 미연시처럼 기본 화면이 보여야 해
- 화면 전체에 배경 이미지 영역이 있어야 해
- 아래쪽에 대사창이 있어야 해
- 대사창 위나 안쪽에 이름창이 있어야 해
- 대사창 오른쪽 아래에 다음 진행 안내 텍스트가 있어야 해
- 나중에 캐릭터 초상화와 선택지를 넣을 빈 오브젝트 자리도 있어야 해
```
